# Manipulacion-Visualizacion-de-datos
Instacart es una plataforma de entregas de comestibles (similar a Uber Eats pero para supermercados) donde los clientes pueden hacer pedidos online y recibir sus compras a domicilio.

### Objetivo del proyecto:
Analizar los hábitos de compra de los clientes de Instacart para generar insights que ayuden al negocio.

### Datos que trabajados:
- 5 tablas diferentes: pedidos, productos, pasillos, departamentos, y productos por pedido
- Datos reales publicados por Instacart en 2017 para una competición de Kaggle

Principales análisis que realizados:
Patrones temporales:
- Horas del día con más pedidos
- Días de la semana más populares
- Tiempo entre pedidos

Comportamiento de compra:
- Productos más populares
- Productos más reordenados
- Cantidad de artículos por pedido
- Primeros productos que la gente agrega al carrito

### Habilidades técnicas demostradas:
- Limpieza y preprocesamiento de datos
- Análisis exploratorio de datos (EDA)
- Creación de visualizaciones
- Trabajo con múltiples tablas relacionadas
- Identificación de patrones de comportamiento del consumidor

Este proyecto muestra análisis de datos reales de una empresa conocida y demuestra habilidades muy valoradas en el mercado laboral.
